class Host

A generic controller for serving an application. Hosts several class Falcon::Services based on the command configuration.

The configuration is provided by class Falcon::Command::Host and is typically loaded from a falcon.rb file. See Falcon::Configuration#load_file for more details.

def initialize(command, **options)
	@command = command
	
	@configuration = command.configuration
	@services = Services.new(@configuration)
	
	super(**options)
end
